# hamqth-mcp

MCP server for [HamQTH.com](https://www.hamqth.com/) -- callsign lookup (free QRZ alternative).

Part of the [qso-graph](https://qso-graph.io/) collection of amateur radio MCP servers.

## Planned Tools

| Tool | Description |
|------|-------------|
| `hamqth_lookup` | Callsign lookup (name, grid, country, QSL info) |
| `hamqth_recent` | Recent activity for a callsign |
